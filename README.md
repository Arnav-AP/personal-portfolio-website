# Arnav’s Space — Personal Portfolio

_A modern, multi-page portfolio website showcasing projects, achievements, and skills with a dark, glowing card-based UI._

---

## Overview

**Arnav’s Space** is a personal portfolio website built to present my projects, achievements, certifications, and skills in a visually engaging, professional format. The site is fully static, responsive, and designed for easy deployment on GitHub Pages.

---

## Features

- Multi-page static site: `home.html`, `projects.html`, `certifications.html`
- Responsive navbar with accent branding and navigation
- Hero section with personal introduction
- Project showcase: card-based layout for projects with GitHub links
- Achievements & Certifications: detailed cards with certificate links
- Skills and technology stack section
- “Facts About Me” and personal highlights
- Contact section with email, GitHub, LinkedIn, and Discord copy button
- Dark theme with blue accent glow and modern card UI
- Fully responsive (Bootstrap 5 grid)
- Minimal JavaScript for clipboard copy functionality

---

## 🛠 Tech Stack

- HTML5
- CSS3 (custom + Bootstrap 5)
- Bootstrap 5 (CDN)
- Google Fonts (Poppins)
- JavaScript (for copy-to-clipboard)

---

## 🎨 UI / Design Highlights

- **Dark theme**: Black backgrounds for all pages
- **Accent glow**: Blue accent color with glowing text and button effects
- **Card-based UI**: Projects, achievements, and skills are presented in modern cards
- **Responsive grid**: Bootstrap 5 grid system for layout
- **Custom styles**: `styles.css` for accent, glow, and card enhancements
- **Consistent branding**: Profile image and accent color across all pages

## 🖥 How to Run Locally

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-github-username/your-repo-name.git
   cd your-repo-name
   ```
2. **Open `home.html` in your browser.**
   - No build step or server required (fully static site).

---

## 🚢 Deployment (GitHub Pages)

1. Push all files to your GitHub repository.
2. Go to **Settings > Pages** in your repo.
3. Set the source branch to `main` (or `master`) and root directory (`/`).
4. Save and wait for GitHub Pages to deploy.
5. Access your site at `https://your-github-username.github.io/your-repo-name/`.

## 🛠 Customization

- **Edit content:**  
  - Update text, sections, and links directly in `home.html`, `projects.html`, and `certifications.html`.
- **Add/Remove projects:**  
  - Edit the project cards in `projects.html`.
- **Update achievements/certificates:**  
  - Edit cards in `certifications.html` and add certificate files to `certificates/`.
- **Change styles:**  
  - Modify accent colors, glow, and card styles in `styles.css`.
- **Update profile image:**  
  - Replace the image in `images/Arnav Steam Profile Photo.png`.
